# AeviaLaunch

AI-powered website generator. Choose a template, describe your business, get a production-ready site in seconds.

## Features

- 15+ premium templates (Luxury, Brutalist, Aurora, Magazine, 3D-Tech, Minimal Pro + 9 more)
- Claude Haiku AI generates all copy (headline, about, services, testimonials)
- Framer Motion animations: parallax, stagger, magnetic buttons, word-by-word reveal
- Custom brand color, logo upload, hero image
- Instant preview + shareable link

## Stack

Next.js 16 · React 19 · Framer Motion · Anthropic Claude Haiku · TypeScript
